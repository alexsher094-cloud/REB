Ответы на контрольные вопросы
1. Автокорреляционная функция прямоугольного импульса без ЛЧМ
АКФ прямоугольного импульса длительностью τи имеет треугольную форму:

text
B(τ) = 1 - |τ|/τи  при |τ| ≤ τи
B(τ) = 0           при |τ| > τи
Характеристики:

Ширина главного лепестка по нулям: 2τи

Ширина на уровне 0.5: τи

Уровень боковых лепестков: отсутствуют (чисто треугольная форма)

2. Автокорреляционная функция прямоугольного импульса с ЛЧМ
АКФ сигнала с линейной частотной модуляцией имеет вид:

text
|B(τ)| ≈ |sin(πΔfτ(1 - |τ|/τи)) / (πΔfτ)|  при |τ| ≤ τи
Характеристики:

Главный лепесток сжимается до ширины ≈ 1/Δf

Уровень боковых лепестков: ≈ -13.2 дБ (первый боковой)

Происходит сжатие импульса в Δf·τи раз

3. Автокорреляционная функция ФКМ импульса
Для фазоманипулированного сигнала (код Баркера, М-последовательность):

Код Баркера:

Главный лепесток: максимальное значение = N (длина кода)

Боковые лепестки: уровень 1/N (-20·log₁₀(N) дБ)

М-последовательность:

Главный лепесток: N

Боковые лепестки: уровень -1/N (примерно -20·log₁₀(N) дБ)

4. Функция неопределенности (в общем виде)
Функция неопределенности определяется как:

text
|χ(τ, f)|² = |∫ s(t)·s*(t+τ)·e^(j2πft) dt|²
где:

τ - временная задержка

f - доплеровский сдвиг частоты

s(t) - комплексная огибающая сигнала

Свойства:

|χ(0,0)|² = E² (энергия сигнала в квадрате)

|χ(τ,f)|² ≤ |χ(0,0)|²

Объем функции неопределенности постоянен: ∫∫ |χ(τ,f)|² dτ df = E²

5. Диаграммы неопределенности (в общем виде)
Диаграмма неопределенности - это сечение функции неопределенности на заданном уровне (обычно 0.5 или 0.707).

Типы диаграмм:

Тип «нож» (knife-edge) - характерен для ЛЧМ сигналов

Тип «колокол» (bed of nails) - характерен для пачек импульсов

Тип «шило» (thumbtack) - характерен для сложных сигналов

6. Функции неопределенности одиночных сигналов
Прямоугольный импульс:

text
|χ(τ,f)| = |(1 - |τ|/τи)·sin(πf(τи - |τ|))/(πf(τи - |τ|))|
ЛЧМ сигнал:

Имеет наклонный гребень: f = (Δf/τи)·τ

Обеспечивает связь между задержкой и доплеровским сдвигом

7. Функции неопределенности повторяющихся сигналов
Для периодических сигналов ФН имеет гребенчатую структуру:

text
|χ(τ,f)| = |χ₁(τ,f)| * |Σ exp(j2πf·nTpr)|
где χ₁(τ,f) - ФН одиночного импульса.

Особенности:

Появляются пики при τ = k·Tpr

Появляются пики при f = m/Tpr

Возникает неоднозначность по дальности и скорости

8. Функции неопределенности пачки сигналов
Для пачки из N импульсов:

text
|χ(τ,f)| = |χ₁(τ,f)| * |sin(πf·N·Tpr) / sin(πf·Tpr)|
Характеристики:

Главный лепесток по частоте: ширина ≈ 1/(N·Tpr)

Боковые лепестки: уровни определяются функцией sin(Nx)/sin(x)

По задержке: сохраняется структура одиночного импульса

9. Применение диаграмм неопределенности в радиолокации
Оценка разрешающей способности - по ширине главного лепестка

Анализ неоднозначности - по расстоянию между пиками

Выбор зондирующего сигнала - для конкретной задачи

Расчет точности измерений - по крутизне спада

Оценка помехоустойчивости - по уровню боковых лепестков

10. Использование диаграмм неопределенности для характеристики разрешения по τ и Ω
Разрешение по задержке (τ):

Определяется шириной главного лепестка по оси τ

Δτ ≈ 1/ΔF (для сложных сигналов)

Δτ ≈ τи (для простых сигналов)

Разрешение по частоте (Ω):

Определяется шириной главного лепестка по оси f

Δf ≈ 1/T (где T - длительность сигнала)

11. Использование диаграмм неопределенности для характеристики точности по τ и Ω
Потенциальная точность:

Обратно пропорциональна крутизне спада функции неопределенности

σ_τ² ≈ 1/(2·(SNR)·β²), где β - среднеквадратичная ширина спектра

σ_f² ≈ 1/(2·(SNR)·Tₑ²), где Tₑ - эффективная длительность

Соотношение неопределенности Гейзенберга:
σ_τ·σ_f ≥ 1/(4π·SNR)

12. Определение диаграммы неопределенности, ее свойства
Определение: Диаграмма неопределенности - это сечение функции |χ(τ,f)|² плоскостью, параллельной плоскости (τ,f) на заданном уровне (обычно 0.5 от максимума).

Свойства:

Симметричность: |χ(-τ,-f)| = |χ(τ,f)|

Максимум в точке (0,0): |χ(0,0)| = E

Инвариантность объема: ∫∫|χ(τ,f)|² dτ df = E²

При изменении масштаба сигнала по времени объем сохраняется

13. Как площадь диаграммы неопределенности зависит от параметров сигнала?
Теорема неопределенности Вудворда:
Площадь диаграммы неопределенности на уровне 0.5 примерно равна:

S ≈ 1 / (ΔF·T)

где:

ΔF - эффективная ширина спектра

T - эффективная длительность сигнала

Следствие: Увеличивая базу сигнала (B = ΔF·T), можно одновременно улучшать разрешение по дальности и скорости, но площадь диаграммы неопределенности остается постоянной.

14. Что происходит с диаграммой неопределенности при введении внутриимпульсной частотной модуляции?
Изменения:

Главный лепесток сжимается по оси τ в Δf·τи раз

Появляется наклонный гребень: f = (Δf/τи)·τ

Сохраняется объем диаграммы неопределенности

Улучшается разрешение по дальности

Эффект:

Δτ ≈ 1/Δf (было ≈ τи)

Δf ≈ 1/τи (без изменений)

15. Какова диаграмма неопределенности сигнала, заданного в виде последовательности δ-функций?
Для последовательности δ-функций:

text
s(t) = Σ δ(t - n·Tpr)
Диаграмма неопределенности:

По оси τ: пики при τ = k·Tpr

По оси f: пики при f = m/Tpr

Форма: «решетка» с шагом Tpr по τ и 1/Tpr по f

Характеристики:

Разрешение по τ: отсутствует (бесконечная неоднозначность)

Разрешение по f: Δf = 1/(N·Tpr)

16. Какова диаграмма неопределенности сигнала, заданного в виде повторяющихся в бесконечных пределах импульсов с гауссовской огибающей?
Сигнал: гауссовские импульсы, повторяющиеся с периодом Tpr

Диаграмма:

Огибающая: гауссовская форма

Структура: гребенчатая

|χ(τ,f)| = exp(-(τ²/2σ_τ²) - (f²/2σ_f²)) * |Σ exp(j2πf·nTpr)|

Особенности:

Отличное разрешение по частоте (за счет периодичности)

Плохое разрешение по дальности (за счет периодичности)

Неоднозначность по дальности с периодом c·Tpr/2

17. Какова диаграмма неопределенности сигнала, заданного в виде пачки импульсов с гауссовской огибающей?
Сигнал: N гауссовских импульсов

Диаграмма:

По задержке: определяется формой одиночного импульса

По частоте: определяется длительностью пачки

|χ(τ,f)| = exp(-(τ²/2σ_τ²)) * |sin(πf·N·Tpr)/sin(πf·Tpr)|

Характеристики:

Δτ ≈ 2σ_τ (по дальности)

Δf ≈ 1/(N·Tpr) (по скорости)

Боковые лепестки по частоте: уровень -13.5 дБ
